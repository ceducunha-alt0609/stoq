Como cadastrar um novo produto? ▾
Toque no botão + (canto inferior direito) em qualquer tela. Preencha nome, categoria, unidade, quantidade atual e estoque mínimo. O ícone pode ser escolhido na grade de emojis.
Como funciona o alerta de reposição? ▾
Cada item tem um estoque mínimo. O alerta amarelo dispara quando a quantidade fica abaixo de 150% desse mínimo (configurável em Alertas). O alerta vermelho dispara quando a quantidade atinge ou fica abaixo do mínimo. Itens zerados ficam críticos imediatamente.
Meus dados ficam salvos se eu fechar o app? ▾
Sim! Todos os dados são salvos automaticamente no localStorage do seu dispositivo. Eles persistem entre sessões. Para não perder dados ao trocar de dispositivo, use a sincronização via GitHub Gist ou JSONBin na aba Sincronização.
Como instalar como app no celular? ▾
Android (Chrome): Toque no menu ⋮ → "Adicionar à tela inicial".
iOS (Safari): Toque em Compartilhar ↑ → "Adicionar à Tela de Início". Após instalado, funciona como app nativo com ícone na home.
Como funciona a Conferência de Estoque? ▾
A conferência gera um checklist com todos os itens. Você vai marcando cada um à medida que verifica fisicamente. Ao finalizar, o timer de conferência reinicia. Você pode programar a frequência (semanal, quinzenal, mensal) nas configurações de Alertas.
Como registrar uma entrada ou saída? ▾
Vá para a tela Movimentações e toque em +. Selecione o item, tipo (entrada/saída), quantidade e responsável. Você também pode registrar diretamente pela tela de detalhe de um produto (botões verde/vermelho).
O app funciona sem internet? ▾
Sim, 100%! O Stoq é um PWA com Service Worker. Após a primeira visita, todos os arquivos ficam em cache e o app funciona completamente offline. A sincronização com GitHub/JSONBin/Supabase requer conexão, mas o uso diário não.
Posso usar em múltiplos condomínios? ▾
Atualmente o app gerencia um estoque por instalação. Para múltiplos condomínios, a melhor abordagem é usar a sincronização via JSONBin (crie um Bin diferente para cada condomínio) e alternar entre eles pelo painel de Sincronização.
Como exportar os dados para Excel? ▾
Na aba Sincronização → Backup Local você pode exportar um arquivo .json com todos os dados. Para converter para Excel, abra o arquivo JSON e importe em qualquer ferramenta online de JSON→CSV, depois abra no Excel.
